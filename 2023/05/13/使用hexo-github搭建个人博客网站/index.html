使用hexo+github搭建个人博客网站
2023-05-13

本文面向对象

  • 本文不适合github和node.js没听过的小白。
  • 默认读者已经安装好node.js和git并且自己在github创建过仓库

git密钥连接github（做过的可省略）

  1. 可以先到github主页->头像->setting->SSH and GPG keys查看是否已经设置过
  2. 在根目录右键选择git bash Here打开
  3. 输入以下命令设置git全局配置（name为你github的用户名，email为github的邮箱）
    git --config global "name"
    git –config global “email”
    3 .生成密钥
    h-keygen -t rsa -C "Email"
  4. 查看密钥
    打开/c/Users/you/.ssh/id_rsa.pub.文件并复制内容
    到github主页->头像->setting->SSH and GPG keys点击New SSH将复制内容粘贴到key中，title自己填
  5. 查看是否连接成功
    回到git bash Heres输入
    ssh -T git@github.com

若出现你的用户名则连接成功

github创建仓库

  1. 与正常的创建相同，唯一不同的是Repository name需要设置为 用户名/用户名.github.io
  2. 勾选 Add a README file 即可

安装hexo

  1. 回到git bash Here输入
    npm install hexo-cli -g
  2. 输入
    hexo -v

查看是否安装成功

配置hexo

  1. 依次输入以下命令
    hexo init
    npm install
